#  EcoRoots

EcoRoots is a sustainability-focused web platform designed to promote eco-friendly practices and spread awareness about environmental responsibility. The project focuses on clean UI design, accessibility, and responsive user experience.

---

Features

 Promotes eco-friendly initiatives and awareness  
 Fully responsive design for multiple devices  
 Clean and minimal user interface  
 Structured navigation for better user experience  
 Fast-loading static frontend  

---

## Tech Stack

- HTML5  
- CSS3  
- JavaScript
- NodeJs
